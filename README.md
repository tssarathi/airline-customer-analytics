# Airline Customer Analytics

End-to-end customer analytics and ML pipeline using airline loyalty data.

**Tech stack**
- Python, Pandas
- AWS S3, Glue, Athena
- SageMaker (later)
- QuickSight (later)

## Project Structure
- `src/` – core application code
- `infra/` – infrastructure as code
- `reports/` – dashboards and charts

python -m src
